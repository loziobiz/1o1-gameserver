Welcome to 1o1!

1o1 is essentially a simple game server optimized for card games and for heads-up matches.

Although only the "tresette" game rules are implemented at the moment, with the modular architecture of 1o1, is really easy to bind any other game to the core platform.

You can sign in and try it, using any authentication provider.
A simple angular HTML5 tresette client is boundled and there are always AI players ("bots") waiting in lobby to let you start playing quickly.

The 1o1 platform is built using Mongo, Express, Angular, Node (MEAN) technologies and is an open source project based on the MIT licence.
You can fork it on GitHub, if you want. Instructions to set-up the environment are provided.
